# Glossary — Убиквитарни језик

> Термини се додају само када се идентификују у домену. Никада се не уводе претпоставком.

| Термин | Дефиниција | Bounded Context |
|---|---|---|
| Корисник (User) | Особа са приступом систему; идентификована Firebase Auth налогом; мора бити на whitelist-у | Identity |
| Администратор (Admin) | Корисник са правом уноса, измене и брисања записа; предефинише whitelist налоге | Identity |
| Посматрач (Viewer) | Корисник са искључиво правом читања; не може мењати податке | Identity |
| Whitelist | Скуп email адреса предефинисаних у Firestore-у; одређује ко сме да приступи систему и у ком улогу | Identity |
| Централна каса | Јединствени финансијски фонд организације; сви корисници виде исто стање | Finance |
| Финансијски запис | Атомска јединица евиденције; бележи тачно када, шта, колико, у којој валути, од кога/коме и ко је унео | Finance |
| Приход | Новчани прилив — подтип финансијског записа | Finance |
| Расход | Новчани одлив — подтип финансијског записа | Finance |
| Износ | Value Object; садржи `вредност: number` и `валута: string (ISO 4217)`; нпр. `{ вредност: 5000, валута: "RSD" }` | Finance |
| Валута | ISO 4217 код валуте (нпр. `"RSD"`, `"EUR"`, `"USD"`); саставни део Value Object Износ | Finance |
| Категорија | Класификациона ознака финансијског записа (нпр. „Закуп", „Плате", „Приход од продаје"); постоји за и приходе и расходе; може се додавати, уређивати и меко брисати | Finance |
| Системска категорија | Категорија са заставицом `Системска = true`; не може се брисати ни преименовати; пример: „Непознато" | Finance |
| Непознато | Обавезна системска категорија (по један унос за Приход и Расход); гарантује да сваки запис увек има валидну категорију | Finance |
| Аутор записа | Корисник (Admin) који је унео конкретни финансијски запис; чува се уз запис | Finance |
| Контрагент | Особа или организација од које потиче или којој одлази новац (поље „од кога/коме") | Finance |
| Дашборд | Агрегирани приказ стања Централне касе; подржава филтрирање по периоду, категорији, типу и валути; приказује салдо по свакој валути одвојено | Finance |
| Период | Временски опсег за агрегацију и филтрирање (дан, месец, година, произвољни опсег) | Finance |
| Средство | Именована визуелна резервација новца из Централне касе; нема сопствени баланс — само прати колико је резервисано и колики је капацитет; не урачунава се двоструко у касу | Finance |
| Резервација | Операција премештања новца из слободног дела касе у Средство; извршује само Admin | Finance |
| Слободна средства | Део салда који није резервисан ни у једном Средству = укупни салдо − Σ reserved свих Средстава | Finance |

## Отворени термини (нису потврђени у домену)
- Буџет — могући будући концепт; није у тренутном обиму
